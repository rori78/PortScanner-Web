How to Close Ports

Windows

To close a port on Windows, follow these steps:

  1. Open the Command Prompt as an administrator.
  2. Run the following command to find the process ID (PID) associated with the port: 
    netstat -aon | findstr :PORT_NUMBER
  3. Identify the PID from the output.
  4. Terminate the process using the PID: 
    taskkill /PID PID_NUMBER /F
  5. To permanently close the port, use Windows Firewall to create a new inbound rule that blocks the port.

Linux

To close a port on Linux, follow these steps:

  1. Open a terminal.
  2. Run the following command to find the process using the port: 
    sudo lsof -i :PORT_NUMBER
  3. Identify the process ID (PID) from the output.
  4. Terminate the process using the PID: 
    sudo kill -9 PID_NUMBER
  5. To block the port using iptables, run: 
    sudo iptables -A INPUT -p tcp --dport PORT_NUMBER -j DROP

macOS

To close a port on macOS, follow these steps:

  1. Open a terminal.
  2. Run the following command to find the process using the port: 
    sudo lsof -i :PORT_NUMBER
  3. Identify the process ID (PID) from the output.
  4. Terminate the process using the PID: 
    sudo kill -9 PID_NUMBER
  5. To block the port using pf (Packet Filter), edit the pf configuration file located at /etc/pf.conf and add a rule to block the port, then reload pf: 
    sudo pfctl -f /etc/pf.conf
